# NICA BALANCE

Aplicación móvil para la gestión de tus finanzas personales.

## 📋 Información del Proyecto

- **Nombre**: Nica Balance
- **Versión**: 0.1.0+1
- **Package**: com.alxd.nica_balance
- **SDK Mínimo**: Android 7.0 (API 24)
- **SDK Objetivo**: Android 16 (API 36)

## 🛠️ Tecnologías

- **Flutter**: 3.12.0+
- **State Management**: Provider
- **Base de Datos Local**: ObjectBox

## 🚀 Instalación

### Requisitos Previos

- Flutter SDK 3.12.0 o superior
- Android Studio / Xcode
- Dart SDK

## 📝 Estructura del Proyecto

```
lib/
├── core/           # Base de datos, Constantes, utilidades, theme
├── data/           # Modelos, repositorios
├── presentation/   # Pantallas de la aplicación, View models, Widgets
└── main.dart       # Punto de entrada
```
